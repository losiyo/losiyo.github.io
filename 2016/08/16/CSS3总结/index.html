CSS3总结

CSS3模块

  • 选择器
  • 盒模型
  • 背景和边框
  • 图像值和替换内容
  • 文字特效
  • 2D/3D转换
  • 动画
  • 多列布局

Round Corners 圆角

  • border-radius
  • border-top-left-radius

Border Images(IE 11+)

  • border-image
  • -o-border-image
  • -webkit-border-image
  • border-image-source
  • border-image-slice ——图片边框向内偏移
  • border-image-width ——图片边框的宽度
  • border-image-outset ——边框图像区域超出边框的量
  • border-image-repeat ——平铺(repeated),拉伸(stretched),铺满(rounded)

Backgrounds

  • background-clip ——规定背景的绘制区域，border-box, padding-box, content-box
  • background-image
  • background-origin ——规定背景图片的定位区域, padding-box, border-box, content-box
  • background-size ——auto, length, %, cover, contain, initial, inherit

Colors

  • RGBA, HSL, HSLA, opacity

Gradient 渐变(IE 10+, -o-, -webkit-, -moz-)

  • linear-gradient
  • radial-gradient
  • repeat-linear-gradient
  • repeat-radial-gradient
    background: linear-gradient(direction, color-stop1, color-stop2,…)
    top-to-bottom(default)
    background: linear-gradient(angle, color-stop1, color-stop2,..)
    background: radial-gradient(shape, color…)
    ellipse
    circle
    closest-side
    farthest-side
    closest-side
    farthest-side

阴影效果

  • text-shadow ——IE 10+
  • box-shadow
    参数：水平偏移，垂直偏移，模糊，颜色

Text

  • text-overflow
    ‘’’
    text-overflow: clip;
    ‘’’
    clip: 修剪
    ellipsis: 用省略号代替修剪文本
    string: 用给定字符串代替修剪文本　
  • word-wrap ——允许长单词或URL地址转换到下一行
    normal: 只允许在断点字换行
    break-word: 在长单词或URL地址内部进行换行
  • word-break
    normal
    break-all: 允许在单词内换行
    keep-all: 只能在半角空格或连字符处换行
  • text-align-last
  • text-emphasis
  • text-justify

Web Fonts

@font-face{
font-family:
src:
font-weight:
font-style:
}

2D变换

  • transform ——向元素应用2D或3D变换
  • transform-origin ——改变被变幻元素的位置
    translate(50px, 100px)——平移
    rotate(20 deg)——旋转
    scale(2,3)——缩放
    skew(30deg, 20deg)——绕X轴旋转30度，绕Y轴旋转20度
    skewX(30deg)
    skewY(20deg)
    matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

3D变换 （IE 10+）

  • transform
    matrix3d(…)
    translateX()
    translateY()
    translateZ()
    translate3d(x,y,z)
    scaleX()
    scaleY()
    scaleZ()
    scale3d(x,y,z)
    rotate3d(x,y,z,angle)
    rotateX()
    rotateY()
    ratateZ()
    perspective(n)
  • transform-origin
  • transform-style——规定被前套元素如何在3D空间中显示
    flat: 子元素将不保留其3D位置
    preserve-3d: 子元素将保留其3D位置
  • perspectiv——定义3D元素距离视图的距离，以像素记，其子元素会活的透视效果，而不是元素本身
  • perspective-origin——定义3D元素所基于的X轴和Y轴
  • backface-visibility——当元素不面向屏幕时是否可见
    hidden
    visible

Transition 过渡(IE 10+)

  • transition
  • transition-property
  • transition-duration
  • transition-timing-function
    linear
    ease
    ease-in
    ease-out
    ease-in-out
    cubic-bezier(n,n,n)
  • transition-delay

Animation (IE 10+)

@keyframes

  • animation
  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count——动画被播放的次数，n, infinite
  • animation-direction——下一周期是否逆向播放？normal; reverse; alternate; alternate-reverse;
  • animation-play-state——运行转台pause; running
  • animation-fill-mode——规定动画在之前或之后，其动画效果是否可见？none; forwards: 动画完成后，保持最后一个属性值; backwards: 动画之前，应用属性开始值; both

Multiple Columns 多列布局 (IE 10+)

  • column-count
  • column-gap
  • column-rule
  • column-rule-color
  • column-rule-style
  • column-rule-width
  • columns
  • column-fill
  • column-span
  • column-width
